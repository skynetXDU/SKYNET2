# CHANGELOG

本文件记录 SKYNET 每个版本的重要变化。

## [0.1.0] - 2026-05-07

### 新增
- 新增 `InspectorLabel`，用于自定义 Inspector 字段显示名。
- 新增 `ShowIf` 和 `EnableIf`，用于根据 bool、enum 或 Flags enum 条件显示/禁用字段。
- 新增 `EnumToggleButtons`，用于将枚举绘制成按钮。
- 新增 `TableList` 和 `TableName`，用于将数组或列表绘制成表格。
- 新增 `SDictionary<K,V>` 和 `SDictionaryLabel`，用于在 Unity 中序列化字典并自定义表头。
- 新增示例场景、示例脚本和中文文档。
